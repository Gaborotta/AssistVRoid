AssistVRoid

===

VOICEROID便利ツール ショートカット拡張
CTRL+R : ファイル名を自動でつけて保存
CTRL+0〜CTRL+9 : 音声効果を切り替える

設定は「init.txt」で行う

## 開発環境

Windows7 - VisualStudio2015 C#

なし
